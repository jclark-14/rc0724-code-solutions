# typescript-destructuring-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What is destructuring, conceptually?

- extract values/elements from objects and arrays and assign them to variables

- What is the syntax for `Object` destructuring?

- const { property1, property2 } = object

- What is the syntax for `Array` destructuring?

- const [ element1, element2, etc ] = array

- How can you tell the difference between destructuring and creating `Object`/`Array` literals?

- when creating the object/array is named before it is defined, when destructuring the values, properties come first then assigned to the object name.
